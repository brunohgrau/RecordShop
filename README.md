# RecordShop

### Tools used

- Axios 
- React
- Redux 
- ReactQuery
- Tailwind CSS
- Node 
- Express
- MongoDB

### User Stories

- User and Admin User can login and register
- User can add recrods to cart and  make records order
- Admin User can CRUD Records and Users
